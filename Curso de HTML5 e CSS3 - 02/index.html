Seletores CSS

Id

Possui maior prioridade sobre as classes.
Ele é único. Para utilizar IDs, deve-se utilizar o: #

Classes

Possui menor prioridade em relação aos IDs.
Pode ser quantas vezes for necessário, e em diferentes tags.
É possível aplicar mais de uma classe na mesma tag.
Vale ressaltar que é possível utilizar ID e Classe ao mesmo tempo.
Para utilizar classes, deve-se utilizar o: .

Display

Compared to display: inline, the major difference is that display: inline-block allows to set a width and height on the element.
Also, with display: inline-block, the top and bottom margins/paddings are respected, but with display: inline they are not.
Compared to display: block, the major difference is that display: inline-block does not add a line-break after the element, so the element can sit next to other elements.

Responsividade

Responsividade com largura fixa e alinhamento do conteúdo.

Primeiramente determine a largura mínima da tela. Geralmente, celulares mais antigos possuem uma largura de 320px. Para isso, utilize: min-width: 320px; Dessa forma, a tela terá no mínimo 320px, nada menos que isso.

Posteriormente, determine o tamanho máximo da tela, independente do tamanho da tela. Ex.: max-width: 800px Dessa forma, o elemento que possuir essa propriedade CSS, não terá uma largura maior que 800px, independente do quão larga seja a tela.

Por fim, para centralizar o conteúdo caso a largura do elemento seja menor que a tela, utilize margin: 0 auto

Vale ressaltar que, é possível utilizar as tags picture e source, para tornar a imagem mais reponsiva, assim quando a tela possuir determinado tamanho será exibido uma imagem que adapta-se melhor a essa determianda largura.
