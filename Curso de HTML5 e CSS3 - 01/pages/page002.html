Para retornar, basta clickar aqui

Atributos utilizados nesse link: <a href="../index.html" rel="prev"> </a>
Através do atributo "rel", é possível especificar o tipo do link, se é interno ("next", "prev"), externo, etc.
Neste caso, é um link interno que aponta para a página anterior, através do atributo "rel=prev".
Além disso, há o atributo "target="_self"", no qual especifica que o link aponta para a própria página.

Usando links externos

Link para o GitHub

Atributos utilizados nesse link: <a href="https://github.com/" target="_blank" rel="nofollow"> </a>
Através do atributo "rel", é possível especificar o tipo do link, se é interno ("next", "prev"), externo, etc.
Neste caso, é um link externo, no qual é especificado para o mecanismo de busca não perpetuar a busca, não possui o aval, isso deve-se ao atributo "rel=nofollow".
