Principais Formatações

Negrito/Destaque

TAG <b> </b> - Texto em negrito, apenas formato.

TAG <strong> </strong> - Texto com valor semântico, de destaque.

Itálico/Ênfase

TAG <i> </i> - Texto em itálico, apenas formato.

TAG <em> </em> - Texto com valor semântico, de ênfase.

Geralmente para a formatação de textos, sem o valor semântico, utiliza-se CSS para estilizar.

Texto Marcado

TAG <mark> </mark> - texto marcado

Texto Pequeno

TAG <small> </small> - texto pequeno

Texto Deletado

TAG <del> </del> - texto excluído

Texto Inserido

TAG <ins> </ins> - Texto marcado como inserido, dando ênfase e indicando que ele foi adicionado depois. (POSSUI VALOR SEMÂNTICO)

TAG <u> </u> - Texto sublinhado, não possui valor semântico.

Texto Sobrescrito

TAG <sup> </sup> - Tag sup - para adicionar valores especiais como: x20+3

Texto Subscrito

TAG <sub> </sub> - Tag sub - para adicionar valores especiais como: H2O

Trabalhando com listas

Lista Ordenada

  1. Item 1;
  2. Item 2;
    1. Item 3;
    2. Item 4;
    3. Item 5;

Através do atributo "type", é possível alterar o tipo da lista, para numérico ou letras. Já com o "start", é possível escolher em qual número/letra a lista irá iniciar.
Atributos utilizados nessa lista: <ol type="A" start="3"> </ol>

Lista Não Ordenada

  • Item 1;
  • Item 2;
  • Item 3;
    • Item 4;
    • Item 5;

Através do atributo "type", é possível alterar o tipo da lista, para círculos, quadrados, etc.
Atributos utilizados nessa lista: <ul type="circle"> </ul>

Usando links externos

Link para o perfil no GitHub

Atributos utilizados nesse link: <a href="https://github.com/ssalnum" target="_blank" rel="external"> </a>
Através do atributo "target" e passando o valor "_blank", permite que ao usuário clickar no link, automaticamente abra uma nova aba e redireciona ele ao link externo. Também há o atributo "rel", no qual é responsável por especificar o tipo do link, se é intertno ("next", "prev"), externo, etc.

Usando links internos

Para ir para a página 002, basta clickar aqui.

Atributos utilizados nesse link: <a href="pages/page002.html" rel="next"> </a>
Através do atributo "rel", é possível especificar o tipo do link, se é intertno ("next", "prev"), externo, etc.
Neste caso, é um link interno que aponta para a próxima página, através do atributo "rel=next".
Além disso, há o atributo "target="_self"", no qual especifica que o link aponta para a própria página.

Imagem Dinâmica

Tente abrir em diferentes resoluções:

TAG <picture> </picture> - O elemento HTML picture é um container usado para especificar múltiplos elementos source para um elemento específico img contido nele.

TAG <source> </source> - O elemento source é utilizado para especificar múltiplos recursos de mídia de elementos picture, audio ou video em HTML5.

<source media="(max-width: 750px)" srcset="imgs/foto-p.png" type="image/png" > </source>
A imagem disponibilizada no path da tag source, é exibida assim que à página ultrapassar 750px, que é estabelecido no atributo "media="(max-width: 750px)""

<source media="(max-width: 1000px)" srcset="imgs/foto-p.png" type="image/png"> </source>
A imagem disponibilizada no path da tag source, é exibida assim que à página ultrapassar 1000px, que é estabelecido no atributo "media="(max-width: 1000px)""

Reproduzindo áudio

Atributos utilizados na tag audio: <audio> src="midia/happy-mistake.mp3" controls </audio>
Para adicionar um audio, basta utilizar a tag "audio", juntamente com o atributo "src=path/do/audio" e para mostrar os controles, utilize o atributo "contorls"

Inserindo vídeos hopedados localmente

Vale destacar que, a ordem dos "source" dentro da tag "vídeo", influencia na tentativa de reprodução, seguindo o a ordem utilizada, seria:
mp4 -> m4v -> webm -> ogv 

          
            <video width="500" controls 
              <source> src="midia/meu-video.mp4" type="video/mp4"</source>
              <source> src="midia/meu-video.m4v" type="video/m4v"</source> 
              <source> src="midia/meu-video.webm"type="video/webm"</source>
              <source> src="midia/meu-video.ogv" type="video/ogg"</source>
            </video>
